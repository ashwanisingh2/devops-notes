# 🚀 God Mode DevOps Vault

Welcome to the **God Mode DevOps Vault**, a production-grade, bilingual (English + Hindi) knowledge base designed for engineers transitioning from L1 Support to Senior Site Reliability Engineer (SRE) roles.

This repository is structured as an **Obsidian** workspace, containing meticulously crafted notes that cover real-world enterprise scenarios, desi analogies for complex concepts, and practical step-by-step labs.

---

## 🌟 Key Features
- **Bilingual Content:** Deep technical concepts explained in English with relatable Hindi analogies (*Desi touch!*).
- **Practical Labs:** Step-by-step labs with actual commands that work when copied, along with expected outputs.
- **Cheat Sheets:** Quick reference tables for the commands you'll actually use on the job.
- **Troubleshooting Guides:** Real-world problems, their likely causes, and step-by-step fixes.
- **Interview Q&A:** Over 90+ targeted interview questions and answers (including STAR format for scenario questions).
- **No Fluff:** Straight to the point, avoiding unnecessary jargon, placeholders, or filler content.

## 🚀 How to Use This Vault

To get the best experience, view these files using Obsidian:

1. Download and install [Obsidian](https://obsidian.md/).
2. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/ashwanisingh2/devops-notes.git
   ```
3. Open Obsidian and select **"Open folder as vault"**.
4. Choose the `devops-notes` folder you just cloned.
5. Start your journey by opening the `00-MOC/Master-Index.md` file!
