# Security Policy

## Supported Versions

Currently, the `main` branch is the only supported version.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project (e.g., exposed secrets in examples, insecure recommended configurations), please do not open a public issue.

Instead, please send an email to the repository owner or use GitHub's private vulnerability reporting feature if enabled. We will aim to acknowledge the issue within 48 hours and release a patch as soon as possible.
